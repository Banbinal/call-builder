# Skill `reecriture-ticket`

Transforme un ticket flou en ticket exécutable : hypothèses explicitées et
marquées `[HYPOTHÈSE]`, cas limites, sections fixes (Contexte / Objectif /
Spécifications / Critères d'acceptation testables / Hors scope), questions
ouvertes en fin de réponse.

## Installation en 3 lignes

1. Copier ce dossier dans `~/.claude/skills/` (Windows :
   `%USERPROFILE%\.claude\skills\`).
2. Ouvrir une session Claude Code.
3. Coller votre ticket avec « réécris ce ticket : … » — la skill se déclenche.

Suite d'évals (5 tickets flous, 3 critères binaires chacun) dans
[`evals/`](evals/).
